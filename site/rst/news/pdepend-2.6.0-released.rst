=======================
Version 2.6.0 released
=======================

We are proud to announce version `2.6.0 <https://github.com/pdepend/pdepend/releases/tag/2.6.0>`_ of PDepend. For more
details visit the version release `site <https://github.com/pdepend/pdepend/releases/tag/2.6.0>`_.

Use `composer <http://getcomposer.org>`_ to install PHP_Depend:

.. class:: shell

::

  curl -s http://getcomposer.org/installer | php
  php composer.phar require "pdepend/pdepend:2.6.0"

Or if you already have composer installed globally:

.. class:: shell

::

  composer require "pdepend/pdepend:2.6.0"
